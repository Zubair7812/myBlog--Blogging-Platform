### Clone the Repository

```bash
git clone https://github.com/Zubair7812/myBlog-Blogging-Platform.git
cd Blogging-Website
```
### Install dependencies
```console 
npm install
```
### Start
```console
node src/App.js
```
```sh
The Website will be available at http://localhost:3000
```
